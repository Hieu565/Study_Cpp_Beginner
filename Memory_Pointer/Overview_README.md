# Overview

This folder contains multiple C++ projects focused on practicing functions and building a solid understanding of basic memory and pointer concept. Each project explores different ways to use Pass-By-Reference to modify the variable within the function, access memory location and help save computation cost of making an argument.

The goal of this collection is to strengthen my problem-solving skills and gain hands-on experience by working on a variety of small to medium-sized programs. Each project has its own folder with a detailed README explaining its purpose and implementation.
