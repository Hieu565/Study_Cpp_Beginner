# Overview

This folder contains multiple C++ projects focused on practicing functions and building a solid understanding of basic functions concept. Each project explores different ways to use vector for structuring code, improving readability, and dynamic-array functionality.

The goal of this collection is to strengthen my problem-solving skills and gain hands-on experience by working on a variety of small to medium-sized programs. Each project has its own folder with a detailed README explaining its purpose and implementation.
